Как открыть сайт?

1) Скачиваем Python. При установке нажимаем галочку "Добавить в PATH".

2) После установки заходим в командную строку. Пишем python. Если он найден, все хорошо.

3) Нужно установить Django и Pillow. Для этого сначала вводим "py -m pip install Django". После установки Django вводим "py -m pip install Pillow".

4) После в командной строке с помощью команды "cd" переходим в папку, где лежат файлы сайта (заходим внутрь my_site). Затем прописываем py manage.py runserver.

5) Получаем адрес (http://127.0.0.1:8000/). Копируем эту ссылку и открываем в браузере наш сайт. Сервер сайта активен пока открыта командная строка.
